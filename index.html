🔧 Firebase Setup Guide

One-time setup (5 minutes). Create a free Firebase project to enable cloud sync.

1
Create Firebase ProjectGo to console.firebase.google.com → Add project → Name it "Nishat Homeo"
2
Enable Google Sign-InAuthentication → Sign-in method → Google → Enable → Save
3
Create Firestore DatabaseFirestore Database → Create database → Start in production mode → Choose region (asia-south1) → Done
4
Add Firestore Security RulesFirestore → Rules → Replace with:rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth.uid == userId; } } }
5
Get Firebase ConfigProject Settings → Your apps → Add app (Web) → Register → Copy the config values below
6
Add Authorized DomainAuthentication → Settings → Authorized domains → Add your GitHub Pages URL (e.g. yourname.github.io)

Paste your Firebase config values:

Config is saved locally in your browser. You only need to do this once per device. The app works offline with local storage even without Firebase.
